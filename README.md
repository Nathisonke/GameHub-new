# GameHub

GameHub is a video game discovery web app that helps you find new and interesting games to play. With GameHub, you can search for games by platform, genre, and more. 

You can access the deployed app at : https://game-hub-blond-pi.vercel.app/

## About the App and how it is built:

 - it gives the user different kind of game genres
 - Platforms where you can play the game.
 - Order wich is curreted by:
 -  1. Relevence
    2. Date added
    3. Name
    4. Release date
    5. Popularity
    6. Avarage rating
  
 -  The API used during the creation  of the game comes from : https://rawg.io/apidocs
